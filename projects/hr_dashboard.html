HR Analytics Dashboard

A Tableau dashboard analyzing workforce attrition, satisfaction, and diversity metrics

← Previous Project Back to Portfolio

Project Overview

This Tableau dashboard provides comprehensive HR insights by analyzing employee demographics, attrition patterns, and satisfaction levels. Designed to support data-driven decision-making, it helps organizations identify retention risks and optimize workforce strategies.

The dashboard enables HR teams to:

  • Track key attrition metrics and trends
  • Analyze demographic distributions
  • Identify high-risk departments and roles
  • Evaluate job satisfaction across teams

Key Features & Analysis

  • Attrition Analysis

    R&D department shows highest attrition, with Sales Executives and employees aged 55+ being most affected.

  • Demographic Trends

    Bachelor's degree holders dominate the workforce, while males exhibit higher attrition rates than females.

  • Satisfaction Paradox

    Despite high attrition, Sales Executives report strong job satisfaction, suggesting non-satisfaction drivers.

  • Diversity Metrics

    Breakdowns by gender, race, and marital status reveal opportunities for improved inclusivity.

  • Strategic KPIs

    Tracked attrition rate, average age, active employees, and department-specific trends.

Technical Implementation

The dashboard was built with:

  • Tableau for interactive visualizations
  • Excel for initial data cleaning and pivoting
  • Advanced calculations for attrition rate formulas
  • Parameter controls for dynamic filtering
  • Dashboard actions for cross-chart interactivity
