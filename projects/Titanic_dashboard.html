Titanic Data Analysis

An Excel analysis of passenger survival patterns and demographics

Project Overview

This Excel-based analysis explores survival patterns and passenger demographics from the Titanic disaster. Using pivot tables, charts, and conditional formatting, it reveals how gender, class, and age influenced survival rates during this historical event.

The analysis enables viewers to:

  • Understand survival disparities across passenger groups
  • Compare boarding locations and fare distributions
  • Identify demographic patterns in the passenger manifest
  • Visualize historical data through Excel's analytical tools

Key Features & Analysis

  • Gender Survival Gap

    Over 80% of men did not survive, while women comprised 65% of survivors, highlighting evacuation priorities.

  • Class Disparities

    3rd Class passengers faced the highest fatality rate (67%), revealing socioeconomic impacts on survival.

  • Geographic Trends

    Most passengers boarded from Southampton, with embarkation location correlating to class distribution.

  • Fare Analysis

    1st Class tickets commanded premium prices, with fare ranges exposing stark class differences.

  • Age Insights

    Children showed marginally better survival rates, though "women and children first" wasn't uniformly applied.

Technical Implementation

The analysis was built with:

  • Excel data cleaning and validation techniques
  • Pivot tables for multi-dimensional analysis
  • Conditional formatting to highlight critical metrics
  • Custom charts for comparative visualization
  • Data storytelling principles to convey historical context
