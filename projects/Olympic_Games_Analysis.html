Project Overview

This Power BI dashboard provides historical insights into Olympic Games performance by analyzing key metrics including athlete statistics, medal distributions, and country rankings over time. The project combines Python data scraping with Power BI visualization to create a comprehensive view of Olympic history.

The dashboard enables Viwers to:

  • Understand athlete performance trends across different sports and events
  • Analyze country dominance patterns in specific sports
  • Track medal distributions by gender, age, and country
  • Compare Summer and Winter Games participation

Key Features & Analysis

  • Athlete Analytics

    Tracking medal counts, age distributions, and physical attributes of athletes to identify performance patterns across different sports and events.

  • Country Performance

    Calculating total medals and medal percentages by country with visualizations showing historical trends and sporting strengths.

  • Data Enrichment

    Enhancing raw NOC codes with full country names and scraping additional athlete data from Olympedia.org for more comprehensive analysis.

  • Historical Trends

    Visualizing participation growth, gender equality progress, and sporting event evolution across Olympic history.

  • Sport-Specific Insights

    Identifying dominant countries in specific sports and tracking performance changes over multiple Olympiads.

Technical Implementation

The dashboard was built in Power BI with:

  • Data modeling to connect athlete, event, and country tables
  • DAX measures for calculating KPIs like medal percentages and country rankings
  • Interactive filters and slicers for dynamic analysis
  • Custom visualizations to highlight key relationships
  • Python scripts for data scraping and enrichment
